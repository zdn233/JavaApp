Dainan Zhang -- Android Fotag App

important instructions:
1. When the app is launched, there's no image shown. The user have to click the "load" button to load images from "https://www.student.cs.uwaterloo.ca/~cs349/s19/assignments/images".

2. The "clear" button clears the current images and all ratings (also the filter rating).

3. The "load" button loads all images from "https://www.student.cs.uwaterloo.ca/~cs349/s19/assignments/images". All the images loaded has rating of 0. Click on this button also clears all the previous ratings and filter rating.

4. To reset ratings(either image rating or filter rating), just put 0 star in the "Rating Bar", then the rating is set to 0. After that the user can choose a new rating. This is how the user reset or change the previous ratings.
